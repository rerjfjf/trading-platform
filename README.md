# Algorithmic Trading Research Platform

Платформа для разработки, тестирования и анализа торговых стратегий.

## Технологии
- Python 3.12 — основная логика
- Rust — высокопроизводительный движок
- PostgreSQL — хранение данных
- FastAPI — бэкенд
- React — интерфейс

## Возможности
- Загрузка реальных данных акций
- Бэктестинг торговых стратегий
- Математические модели (Блэк-Шоулз, Марковиц)
- ML прогнозирование (LSTM)
